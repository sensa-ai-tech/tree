## 📋 Phase 開始提醒（每週發送）

### Phase 1 開始提醒

```
📋 Phase 1 開始：知識模型審核

各位夥伴，本週開始第一階段實測！

🎯 任務：審核心臟科（CARD）知識樹骨架
⏱️ 預計時間：30 分鐘
📖 指南：docs/TESTING_PLAN.md 第三章

操作步驟：
1. git pull 取得最新程式碼
2. npm run dev 啟動
3. 前往 /graph 查看知識圖譜
4. 對照評估表進行評分
5. 在 GitHub 建立 Issue（使用「知識模型反饋」模板）

反饋模板連結：
https://github.com/sensa-ai-tech/tree/issues/new?template=model-feedback.md

請於本週五前完成，感謝！🙏
```

### Phase 2 開始提醒

```
📖 Phase 2 開始：內容品質評估

各位夥伴，本週開始第二階段！

🎯 任務：試讀 3 個範例節點的 AI 生成教學內容
⏱️ 預計時間：45 分鐘
📖 指南：docs/TESTING_PLAN.md 第四章

重點評估：
• 醫學正確性（事實、藥物劑量）
• 台灣在地化（藥物可取得性、盛行率）
• ACVIM 分期等標準是否為最新版

反饋模板連結：
https://github.com/sensa-ai-tech/tree/issues/new?template=content-feedback.md

請於本週五前完成，感謝！🙏
```

### Phase 3 開始提醒

```
🎮 Phase 3 開始：互動功能測試

各位夥伴，進入最後的互動測試階段！

🎯 任務：體驗測驗引擎和病例闖關
⏱️ 預計時間：30 分鐘
```
